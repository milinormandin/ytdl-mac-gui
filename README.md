# ytdl-mac-gui
A simple graphical user interface for the video download CLI tool, [youtube-dl](https://github.com/ytdl-org/youtube-dl). Built exclusively for macOS


## Dev Setup

Activate virtual environment
`source .venv/bin/activate`
